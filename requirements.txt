# Ollama 安装与模型下载

# 1. 下载并安装 Ollama（Windows）
#    https://ollama.com/download/windows
#    或直接运行安装包，安装后 Ollama 会作为系统服务自动启动。
#
# 2. 拉取本项目使用的模型
#    在终端（PowerShell / CMD）运行：
#
#      ollama pull qwen3:8b
#
#    qwen3:8b 约 5.2 GB，首次下载需等待。
#
# 3. 确认 Ollama 正在运行
#
#      ollama list
#
#    应看到 qwen3:8b 在列表中。
#    Ollama 服务地址默认为 http://localhost:11434
#
# 4. 可选：查看模型 context 配置
#
#      ollama show qwen3:8b
